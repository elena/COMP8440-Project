Hi Elena,

As you may or may not be aware, the Linux Conference of Australia
(linux.conf.au or LCA) is to be held at ANU in January 2013 (next year).

LCA uses a web-based registration and overall organisation system
called zookeepr, which is written in Python. Each year the organising
committee (which changes each year as the conference location changes)
makes some changes to zookeepr to better reflect the way they want to
organise the conference.

Anyway, the LCA2013 conference organisers are organising a hackfest
on zookeepr sometime in April. Are you/the Python user group at all
interested in helping out with the hacking of zookeepr?

I'll let Mikal (Michael) communicate more directly with you about what
it is they are hoping to achieve with the hacking of zookeepr, if you
are interested in getting involved.
